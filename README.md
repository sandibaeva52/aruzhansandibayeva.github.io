# NuxtJs Tailwind Blog

This is a nuxtjs, Tailwind CSS blogging template. Its supported Markdown for blog

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

```
